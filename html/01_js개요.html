Javascipt

scipt 언어

    기본 프로그램 (ex. 브라우저)의 동작을 
    사용자의 요구에 맞게 수행되도록 해주는 용도의 언어
    
    별도의 소스코드 컴파일을 하지 않고 
    인터프리터를 이용해 코드를 바로 실행.


    -장점 : 컴파일에 소요되는 시간이 없다  

    -단점 : 프로그램 실행 시간이 상대적으로 느림


    *컴파일 (compile) == 번역가
    - 작성된 코드 파일을 한 번에 이진수로 번역하는 과정.
    
    *인터프리터 (interpreter) == 통역가
    - 한 줄씩 코드를 해석해서 바로 수행

Javascipt란?

    웹 브라우저에서  (크롬, 엣지, 오페라, 파이어폭스,사파리)
    주로 사용하는 인터프리터 방식의 객체 지향 프로그래밍 언어.

    ECMA  script 표준을 따른는 대표적인 웹 기술
    (JS 버전 이름 : ESS,ES6, ....)
